# Codebase Skills

Skills for understanding existing repositories, identifying risk, and planning where to read or change code first.

| Skill | Description |
| --- | --- |
| [git-history-audit](git-history-audit/) | Audit Git history for churn hotspots, ownership concentration, bug clusters, activity trends, and firefighting patterns before reading code. |
| [legacy-codebase-audit](legacy-codebase-audit/) | Audit unfamiliar or legacy codebases across languages and frameworks to map risk, architecture, tests, operations, and first actions. |
